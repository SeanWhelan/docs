---
title: "How to Generate an Estuary Flow Refresh Token"
description: "To generate a Refresh Token, navigate to the Admin page, then head over to the CLI-API section."
---

Press the Generate token button to bring up the modal where you are able to give your token a name. Choose a name that you will be able to use to identify which service your token is meant to give access to.
